Random Password Generator — это графическое приложение для генерации безопасных случайных паролей. Программа позволяет настраивать длину пароля и выбирать типы используемых символов. Все сгенерированные пароли сохраняются в историю с указанием даты и времени создания.

- Генерация паролей длиной от 4 до 64 символов
- Выбор типов символов: цифры, буквы, специальные символы
- Копирование пароля в буфер обмена
- Сохранение истории паролей в JSON файл
- Загрузка истории при запуске программы
- Очистка истории
  Как запустить программу :
python main.py
